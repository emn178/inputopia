Date

jquery-ui

jQuery UI datepicker for date input.

Source Code

          <input class="form-control" name="date" type="date" value="2016-01-01" 
            data-options='{"dateFormat": "yy-mm-dd"}' />

Result

bootstrap-datepicker

bootstrap-datepicker for date input. You can use data-vendor to specify.

Source Code

          <input class="form-control" name="date" type="date" value="2016-01-01"
            data-vendor="bootstrap-datepicker" data-options='{"format": "yyyy-mm-dd"}' />

Result

Browser Default

You can use data-norender if you want to use browser default input.

Source Code

          <input class="form-control" name="date" type="date"
            value="2016-01-01" data-norender />

Result

Datetime

bootstrap-datetimepicker

bootstrap-datetimepicker for datetime input. You can set a default options for specific type.

Source Code

          };

Color

kolor-picker

kolor-picker for color input. You can also set options by using JavaScript.

Source Code

          <input id="color" class="form-control" name="color" type="color"
            value="rgba(0,0,0,1)" />
          <span id="color-text"></span>
        
          $(document).ready(function () {
            $('#color').data('options', {
              onSelect: function (color) {
                $('#color-text').text(color);
              }
            });
          });

Custom Plugin

You can register your custom plugin and reuse it.

Source Code

          inputopia.register('myplugin', 'city', {
            available: function () {
              return $.fn.autocomplete;
            },
            render: function (options) {
              options.source = function (request, response) {
                $.ajax({
                  url: "http://gd.geobytes.com/AutoCompleteCity",
                  dataType: "jsonp",
                  data: {
                    q: request.term
                  },
                  success: function (data) {
                    response(data.length === 1 && data[ 0 ].length === 0 ? [] : data);
                  }
                });
              };
              options.minLength = 3;
              $(this).autocomplete(options);
            }
          });
        
          <input class="form-control" name="city" type="city" />

Result
